FULL BELLだったら鳥なのに

BELL : /

TECHNICAL SCORE :

エラー

TECHNICAL SCORE(理想)

-

  • SSSまで : - -> -
  • SSS+まで : - -> -

詳細

  • ベルによる合計失点 : -
  • ベル1つあたりの配点 : -

スコア内訳

  • ベルによるスコア : - / 60000 ( MAX- - | -%)
  • ノーツによるスコア : - / 950000 ( MAX- - | -%)

説明・使い方

SEGAのアーケード音楽ゲーム「オンゲキ」に関する計算ツールです。

FULL BELLを逃してしまったとき、もしFULL BELLだったらどのくらいのスコアになるのか計算できます。

BELL欄に獲得ベル数と総ベル数、TECHNICAL SCORE欄にFULLBELLを逃した点数を入力してボタンを押してください。
